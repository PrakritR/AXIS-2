# AXIS-2

Axis Housing — Next.js frontend scaffold: public marketing site, auth screens, and **Manager**, **Resident**, and **Admin** portals (UI and navigation only; mock data).

## Develop

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```

Built with [Next.js](https://nextjs.org).
